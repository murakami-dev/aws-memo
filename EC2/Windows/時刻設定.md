# 以下はドキュメントに記載のとおりの構成
## タイムゾーンの変更
```
## まずは利用可能なすべてのタイムゾーンの一覧を表示
PS C:\Users\hiroya> tzutil /l
(UTC+09:00) Osaka, Sapporo, Tokyo
Tokyo Standard Time

## タイムゾーン変更 **管理者権限が必要。設定後直ちに反映**
PS C:\Windows\system32> tzutil /s "Tokyo Standard Time"
PS C:\Windows\system32>
```
