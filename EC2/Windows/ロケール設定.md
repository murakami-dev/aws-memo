# 確認
```
Get-WinSystemLocale
```

# 設定
- >管理者権限を持つユーザーのみが変更できます。変更は、コンピューターの再起動後に有効になります。
- https://docs.microsoft.com/ja-jp/powershell/module/international/set-winsystemlocale?view=win10-ps
```
Set-WinSystemLocale ja-JP
Set-WinHomeLocation -GeoId 0x7A
```
